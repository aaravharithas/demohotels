# 🏨 Demo Hotels API

A simple **Hotel Management API** built using **Django** and **Django REST Framework (DRF)**.
This project demonstrates CRUD operations, API routing, serialization, and RESTful design principles — perfect for learning DRF or showcasing backend fundamentals.

🌐 **Live API Base URL:**
`https://demohotelsapi.pythonanywhere.com/`

📌 **Hotels Endpoint:**
`https://demohotelsapi.pythonanywhere.com/hotels/`

---

## 🚀 Features

* View a list of hotels
* Retrieve details of a single hotel
* Create new hotel entries
* Update existing hotel information
* Delete hotels
* Clean and simple RESTful structure
* Built using Django REST Framework
