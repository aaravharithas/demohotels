🔍 API Feature Examples

GET /api/hotels/ Returns all hotels with optional pagination.

GET /api/hotels/5/ Uses `GET` with ID path parameter.

GET /api/hotels/?location=Gurgaon
Performs `icontains` search using Django ORM filters.

GET /api/hotels/?price=2000
Filters hotels with an exact price match.

GET /api/hotels/?min_price=3000&max_price=10000
Filters hotels within a specified price range using price__gte and price__lte filters.
Example Range:
Min Price: 1000
Max Price: 10000
💡 Example:
GET /api/hotels/?min_price=5000&max_price=9000&location=Goa
→ Returns hotels in Goa priced between ₹5,000 and ₹9,000.

GET /api/hotels/?rating=4.5
Filter by exact rating match.

GET /api/hotels/?min_rating=2.5&max_rating=5.0
Filters hotels within a rating range using rating__gte and rating__lte filters.
Example Range:
Min Rating: 2.5
Max Rating: 5.0

GET /api/hotels/?search=Regal Crescent
Performs `icontains` search on both name and location fields.

GET /api/hotels/?limit=5&skip=10
Uses Python slicing on the queryset: [skip:skip+limit].

GET /api/hotels/?order_by=-rating
Sort hotels dynamically using `.order_by()` parameter.

POST / PUT / PATCH / DELETE
Handled via DRF generic views or custom methods.
